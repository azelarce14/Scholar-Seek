================================================================================
                          HOW TO USE SEO HELPER
================================================================================

BASIC USAGE:
1. Include the helper at top of PHP file:
   require_once 'includes/seo_helper.php';

2. Add meta tags in <head>:
   <?php outputSEOMetaTags('login'); ?>

3. (Optional) Add structured data:
   <?php echo SEOHelper::generateStructuredData('WebPage', [
       'title' => 'Page Title',
       'description' => 'Page description'
   ]); ?>

AVAILABLE FUNCTIONS:
- outputSEOMetaTags($page, $customData)
- SEOHelper::generateMetaTags($config)
- SEOHelper::generateStructuredData($type, $data)
- SEOHelper::generateBreadcrumbs($breadcrumbs)
- SEOHelper::generateFAQ($faqs)
- SEOHelper::getPageConfig($page, $customData)
- SEOHelper::sanitizeMetaText($text, $maxLength)
- SEOHelper::getCurrentUrl()

PREDEFINED PAGES:
- 'login' - Login page configuration
- 'register' - Registration page configuration
- 'apply' - Scholarship application page configuration
- 'legal' - Legal/Privacy page configuration

                        MONITORING & ANALYTICS

KEY METRICS TO TRACK:

2. Keyword Rankings - Position for target keywords
3. Click-Through Rate (CTR) - % of impressions that result in clicks

6. Registration Conversion - % of visitors who register

8. Bounce Rate - % of visitors who leave without interaction
